旧版本升级

1.0.* 升级到 1.1.*

需要改变的文件：

将 1.1.* 版本中的 config 目录覆盖原先版本，将 1.1.* 中的 app/Exceptions 目录、app/Middleware 目录拷贝到原先版本的 app 目录中。

Request 的变化：

在 1.0.* 中，获取 request 的一些信息是直接通过属性获取的，如：

// 获取 get 参数
$request->get;
// 获取 server 信息
$request->server;

在 1.1.* 中，WorkerA 对属性进行了隐藏，取而代之的是暴露出来的接口，你只需调用方法获取即可：

// 获取 get 参数
$request->get();
// 获取 server 信息
$request->server();

1.0.* 中的文件上传功能比较简陋，直接通过 $request->files 返回了一个文件信息的数组。而 1.1.* 对文件上传提供了更好用的接口，使用方法请参考 基本使用 - 请求。
